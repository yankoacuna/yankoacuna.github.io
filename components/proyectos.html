Proyecto de Titulación

Autoencoder GRU para ECG

IA · Investigación

Jupyter Notebook que detalla el entrenamiento completo del modelo de detección de anomalías utilizado en el Sistema de Visualización ECG. Implementa un Autoencoder con capas GRU (Gated Recurrent Units) para detectar latidos cardíacos anómalos con umbral automático de Otsu.

Resultados y Características
  • Entrenado con dataset PTB-XL (21,837 registros ECG clínicos)
  • Umbral automático de Otsu sobre error de reconstrucción (F1: 0.926)
  • Pipeline completo: filtrado Notch/Pasa Banda, detección picos R, segmentación
  • Doble filtrado de pureza: Correlación + Isolation Forest
  • Comparativa contra LSTM: GRU+Otsu logra Accuracy 87.5%, F1 92.6%
  • Modelo pre-entrenado incluido (model_gru_otsu.pth)
Tecnologías
Python PyTorch Scikit-learn Pandas & NumPy WFDB Matplotlib Seaborn Jupyter Notebook

Detección de Anomalías Cardíacas

IA · Deportistas

Proyecto de Inteligencia Artificial para detectar riesgos cardíacos en deportistas de alto rendimiento a partir de señales ECG de 12 derivaciones. Clasifica ECGs como normales o de riesgo usando KNN con Dynamic Time Warping (DTW), siguiendo el Consenso Internacional de Criterios para la Interpretación del ECG en Atletas.

Funcionalidades y Resultados
  • Clasificación binaria: ECG normal vs. de riesgo (muerte súbita cardíaca)
  • KNN con DTW para comparación de señales multi-derivación de diferente longitud
  • Normalización inter-base y resampleo a 500 Hz de dos datasets (NEADB + LUDB)
  • Extracción de características estadísticas y análisis PCA
  • Recall 100% en casos de riesgo (ningún caso cardíaco de riesgo omitido)
  • Comparativa: KNN-DTW, Decision Tree, Naive Bayes, SVM y XGBoost
Tecnologías
Python Scikit-learn DTW (tslearn) NumPy Pandas WFDB Matplotlib Jupyter Notebook
